**「Movie List」應用程式規劃了兩個頁面**

- 首頁：
  - 顯示資料庫中的所有電影的清單
  - 查看特定電影的詳細資料
  - 搜尋電影
  - 新增喜愛的電影清單
- 收藏頁：
  - 瀏覽喜愛的電影清單
  - 查看特定電影的詳細資料
  - 刪除喜愛的電影清單
